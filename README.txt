Cet outil permet de faire fluctuer le prix de 3 bières selon la demande. Si une bière est commandée son prix augmente de 10% tandis que les autres diminuent de 10%.

Les prix sont affichés par un programme Python : GUY.py
Les "serveurs" peuvent faire fluctuer le prix via une interface web hébergé en local via Python (CALL python -m http.server).
L'interaction entre l'affichage des prix et le site web se fait via requete http ( app.py ).
